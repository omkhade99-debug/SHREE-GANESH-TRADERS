# Shree Ganesh Traders — Website

Product catalogue and contact website for **Shree Ganesh Traders**, a hardware and building materials store located at Modern Shopping Complex, Padoli, Gugus Fata, Chandrapur, Maharashtra.

## Pages
- **Home** — Hero, category tiles, why-us section
- **Products** — Filterable catalogue (Plywood, Doors, Laminates, Hardware, Tools, Adhesives)
- **Contact** — Store info, phone, WhatsApp button, enquiry form

## Tech
Pure HTML + CSS + JS — no frameworks, no build step required.

## Deploy on Netlify
1. Go to [netlify.com](https://netlify.com) → **Add new site → Deploy manually**
2. Drag and drop the `index.html` file (or the whole folder)
3. Done — live in seconds

## Deploy on GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages → Source → main branch / root**
3. Your site will be live at `https://yourusername.github.io/repo-name`
